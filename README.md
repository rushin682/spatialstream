in particular, the [API documentation][].

## Installation

You need to have Python 3.11 or newer installed on your system.
If you don't have Python installed, we recommend installing [uv][].

There are several alternative options to install spatialstream:

<!--
1) Install the latest release of `spatialstream` from [PyPI][]:

```bash
pip install spatialstream
```
-->

1. Install the latest development version:

```bash
pip install git+https://github.com/rushin682/spatialstream.git@main
```
